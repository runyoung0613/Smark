# Smark / 阅记

## 页面布局

所有页面均为Android原生风格（适配手机竖屏，分辨率通用）

启动页
   ↓
首页（文章列表）
├─ 右上角：粘贴链接导入
├─ 系统分享跳转 → 直接解析入库
├─ 搜索文章
└─ 点击文章 → 进入【阅读页】

阅读页
├─ 顶部：刷新/重新抓取正文
├─ 阅读内容 + 长按划线
├─ 底部工具栏：
│   ├─ AI总结 → 弹窗展示
│   ├─ 生成复习题 → 自动创建复习计划
│   └─ 查看我的复习
└─ 返回 → 文章列表

复习中心
├─ 待复习列表（按时间排序）
├─ 点击进入【复习答题页】
└─ 完成后自动更新下一次复习时间

复习答题页
├─ 展示重点原文
├─ 展示AI题目
├─ 作答/查看答案
└─ 标记已复习 → 触发艾宾浩斯时间

设置页
├─ API配置（token/密钥）
├─ 通知开关

## 核心功能

任意网页内容抓取：支持系统分享导入、链接解析、自动提取正文、去掉广告干扰
极简阅读 + 重点划线：长按文本划线标记关键知识点。
AI 智能处理：一键生成文章总结，并基于全文 / 划线自动生成复习题。
艾宾浩斯记忆复习：按记忆曲线自动安排复习计划，手机通知提醒，内置答题复习页面，完成后自动顺延下次复习时间
本地文章管理：文章列表收纳、全文搜索，所有数据存在本地，自用轻量化。

一个能把全网文章抓进来、AI 总结出题、按艾宾浩斯提醒复习的 Android 专属学习阅读工具。

启动页
   ↓

├─ 右上角：粘贴链接导入
├─ 系统分享跳转 → 直接解析入库
├─ 搜索文章
└─ 点击文章 → 进入【阅读页】

├─ 顶部：刷新/重新抓取正文
├─ 阅读内容 + 长按划线
├─ 底部工具栏：
│   ├─ AI总结 → 弹窗展示
│   ├─ 生成复习题 → 自动创建复习计划
│   └─ 查看我的复习
└─ 返回 → 文章列表

复习中心
├─ 待复习列表（按时间排序）
├─ 点击进入【复习答题页】
└─ 完成后自动更新下一次复习时间

复习答题页
├─ 展示重点原文
├─ 展示AI题目
├─ 作答/查看答案

设置页
├─ API配置（token/密钥）
├─ 通知开关

## 2.核心功能

1、任意网页内容导入：链接解析、文字图片导入、去掉广告干扰。
2、阅读协作：长按文本划线，标记关键知识点。
3、AI智能处理：一键生成文章总结，基于划线自动生成复习卡片。
4、艾宾浩斯记忆复习：按记忆曲线自动安排复习计划，手机通知提醒。
5、本地文章管理：文章列表收纳、全文搜索，所有数据存在本地，自用轻量化。
总结：一个能把全网文章抓进来、AI总结复习、按艾宾浩斯提醒复习的 Android 专属学习阅读工具。
